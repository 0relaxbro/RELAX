How this would actually work

  • Read public transaction history for a connected wallet — nothing private, nothing the blockchain doesn't already show anyone.
  • Cross-reference trade timing against market volatility to flag patterns that look like chasing pumps or selling into fear.
  • Turn that into a simple, readable score instead of raw transaction noise — for fun and self-reflection, not financial advice.
  • No connection to the RELAX token contract, its authority, or its supply. This is a separate analysis layer, not a token feature.

Still early. Still testing. We build in public and ship when it's ready.
